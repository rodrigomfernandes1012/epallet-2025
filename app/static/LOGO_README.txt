INSTRUÇÕES PARA ADICIONAR A LOGO DO SISTEMA
=============================================

O arquivo "logo.png" nesta pasta é um placeholder.

Para adicionar a logo da sua empresa:

1. Prepare uma imagem PNG com fundo transparente
2. Tamanho recomendado: 200px de largura (altura proporcional)
3. Substitua o arquivo "logo.png" por sua logo
4. A logo aparecerá na tela de login

Localização do arquivo:
app/static/logo.png

A logo é usada em:
- Tela de login (lado direito)
- Outras telas de autenticação

Formato suportado: PNG (recomendado), JPG, SVG
